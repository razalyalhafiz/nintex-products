# nintex-products

A Vue.js application to select & purchase dummy Nintex products.

## Project commands

### Compiles and minifies for production

```
npm run build
```

### Run the application

```
npm run start
```

### Run unit tests

```
npm run test:unit
```

### Demo application

Click [here](https://nintex-products.herokuapp.com/) to view a running version of the application.
